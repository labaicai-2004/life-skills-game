# 开发需求文档

## 项目概述

**项目名称**：生活小能手（生活技能数字化严肃游戏）

**项目定位**：面向智力障碍儿童的 iPad 交互游戏，帮助学习日常生活技能。

**目标用户**：智力障碍儿童，智力水平相当于正常学龄期 2-4 岁儿童。

**运行平台**：iPad Safari 浏览器（网页应用）。

**线上地址**：https://labaicai-2004.github.io/life-skills-game/

---

## 功能需求

### 核心功能

#### 1. 三关生活技能训练

| 关卡 | 技能 | 步骤数 | 交互方式 |
|------|------|--------|----------|
| 第一关 | 刷牙 | 7 步 | 点击 + 拖拽 |
| 第二关 | 洗脸 | 7 步 | 点击 + 拖拽 |
| 第三关 | 穿衣 | 7 步 | 点击 + 拖拽 |

#### 2. 每关步骤交互

每个步骤包含：
- 语音播报引导（中文女声，语速 0.75x）
- 实物图片展示当前场景（PNG 图片素材）
- 手势交互操作（点击 / 拖拽到目标区域）
- 动态箭头指引（带文字标签 + 圆形底托 + 脉动动画）
- 目标区域可视化（脉动虚线边框 + 发光框标示拖放区）
- 成功反馈（星星粒子动画 + 表扬语音）
- 失败反馈（温和鼓励，不批评不扣分）
- 拖拽重叠阈值 35%（物品覆盖目标超过 35% 即算成功）

#### 3. 交互手势类型

- **点击（tap）**：触摸目标物品
- **拖拽（drag）**：将物品拖到目标区域，重叠超过 35% 即成功

#### 4. 箭头指引系统（7 种动态箭头）

| 箭头类型 | 用途 | 视觉效果 |
|---------|------|---------|
| 点击（脉动圆环） | 指示点击操作 | 橙色圆环缩放闪烁 |
| 向上滑 | 指示向上拖拽 | 白色圆形底托 + 橙色箭头 + 上移动画 |
| 向下滑 | 指示向下拖拽 | 白色圆形底托 + 橙色箭头 + 下移动画 |
| 向左滑 | 指示向左拖拽 | 白色圆形底托 + 橙色箭头 + 左移动画 |
| 向右滑 | 指示向右拖拽 | 白色圆形底托 + 橙色箭头 + 右移动画 |
| 上下滑（双向） | 指示上下搓动 | 双箭头 + 上移动画 |
| 左右滑（双向） | 指示左右擦动 | 双箭头 + 左移动画 |

每个箭头下方有文字标签说明操作（如"拖拽牙刷到嘴边上下刷"）。

### 辅助功能

#### 5. 语音引导系统
- 每步自动播放语音指令
- 成功时随机播放表扬语句（5 种）
- 失败时随机播放鼓励语句（4 种）
- 可一键静音/取消静音
- 优先选择高质量中文女声（Tingting/Sinji/Meijia/Lili）

#### 6. 奖励系统
- 每步完成：星星粒子动画 + 音效
- 整关完成：撒花动画 + 庆祝弹窗 + 音阶音效
- 不做评分，不显示分数

#### 7. 进度展示
- 顶部圆点进度条（7 个点，已完成/当前/未完成三种状态）
- 返回首页按钮
- 音量控制按钮

#### 8. 步骤间连续动画
- 同一关卡内背景保持不变（不闪烁）
- 步骤切换时旧内容淡出、新内容淡入（0.45s 过渡）
- 手势监听在动画完成后才启用

## 非功能需求

### 可用性
- 所有触控目标尺寸不小于 44×44pt（符合 iOS HIG）
- 拖拽判定宽容，重叠 35% 即通过
- 页面禁止缩放，防止误操作
- 无文字依赖，全靠语音+图片引导

### 兼容性
- 目标：iPadOS Safari（iOS 15+）
- 同时兼容桌面端 Chrome/Safari（用于调试）
- 支持横屏和竖屏

### 性能
- 单文件应用，无外部依赖
- 图片文件放置在项目目录内
- 首次加载 < 3 秒（含图片资源）

### 安全性
- 无数据采集
- 无网络请求
- 完全离线可用（部署后通过 Service Worker 缓存）
- 无第三方依赖

## 用户故事

1. 作为**儿童**，我希望能听到温柔的语音告诉我每一步怎么做，这样我不需要认识字也能玩。
2. 作为**儿童**，我希望能看到大大的图画，知道我要碰哪里、往哪边拖。
3. 作为**儿童**，我做错的时候不会被批评，会有声音鼓励我再试一次。
4. 作为**家长/老师**，我希望能一键把游戏装到 iPad 上给孩子用，不需要安装复杂的 App。
5. 作为**家长/老师**，我希望游戏有一个固定的网址，在 iPad 上打开就能玩。

- 不依赖 App Store 审核流程
- 不使用真实儿童照片（隐私考虑）
